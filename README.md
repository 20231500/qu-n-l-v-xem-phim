1. Giới thiệu

Hệ thống giúp quản lý toàn bộ hoạt động bán vé tại rạp phim: từ quản lý phim, suất chiếu, ghế ngồi đến bán vé và thống kê doanh thu.

Hiển thị danh sách phim
🕒 Quản lý suất chiếu
